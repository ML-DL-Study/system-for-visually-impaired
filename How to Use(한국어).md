# 1. install
- 파일을 git을통해 내려받거나 zip으로 받은 후 requirements 를 다운
```
git clone https://github.com/minji-o-j/system-for-visually-impaired.git
cd system-for-visually-impaired
pip install -r requirements.txt
```
